# react-webcam

Webcam component for React

## Installation

```
npm install react-webcam
```

## Demo

http://cezary.github.io/react-webcam/

## Example

```javascript
var React = require('react');
var Webcam = require('react-webcam');

var Component = React.createClass({
  render: function() {
    return (
      <Webcam/>
    );
  }
});
```
